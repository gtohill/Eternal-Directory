== Description ==
The Eternal Resting Directory allows cemeteries to add the deceased to a directory. This directory allows visitors to search the loved ones name. If the name is in the directory, the visitor can select that persons name to view relevant information about the deceased. The visitor can view page dedicated to their loved one. The page includes the options to view an image, an obituiary, the burial location and a map of the cemetery.
